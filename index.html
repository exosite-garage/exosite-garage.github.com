Exosite Garage

Exosite Garage is a github organization that we use to post example projects (listed below) that use Exosite's online services. These projects run on different types of hardware and implement a variety of use cases. Many of the projects are not production ready, but are meant to serve as a starting point to help developers get going.

Embedded and Development Kits

Projects that connect embedded systems and processor development kits to the cloud.
  • Texas Instruments Stellaris Cellular Communications - Projects that connect EK-LM3S6965, EK-LM3S1968 & RDK-IDM-L35 to the cloud via Janus Terminus T2 GPRS cellular modem.
  • Renesas RX62N RDK Micrium - Five Micrium uCOSIII projects that connect RDK peripherals to the cloud and use OEM provisioning to initialize connectivity.
  • Renesas RX62N RDK Renesas Code Base - uIP networking stack project that sends potentiometer data and receives LED control commands to/from the cloud.
  • Renesas RX63N RDK Micrium - A Micrium project that connects RDK peripherals to the cloud and uses OEM provisioning to initialize connectivity.
  • Renesas RL78G13 RDK with GainSpan WiFi module - Connects an YRDKRL78G13 with GainSpan GS1011M WiFi module to the cloud, enabling the device to send on-board sensor data and submit/receive LED controls.
  • Renesas RL78G14 RDK with GainSpan WiFi module - Connects an YRDKRL78G14 with on board GainSpan GS1011M WiFi module to the cloud, enabling the device to send on-board sensor data and submit/receive LED controls.
  • Gateway_Core - Exosite One platform gateway application runs on the Linux box.
  • TI MSP-EXP430FR5739 with CC3000 Smart Config - Connects a MSP-EXP430FR5739 with TI CC3000 WiFi module to the cloud, enabling the device to send on-board sensor data and submit/receive LED controls.
Helper apps
  • Renesas RX62N RDK flash tool - Simple flash tool to recover the MAC address and default Exosite cloud settings.

Arduino Compatible

Projects that run on Arduino (or Arduino-like) hardware.
  • Arduino HTTP POST - Super simple code using Arduino to report to the cloud via Exosite.
  • Fluid Simple Cloud - Uses a Fluid board and the Arduino Exosite library to read/write data via Exosite.
  • Arduino Exosite Library - Puts the Arduino Exosite Library into Arduino IDE to allow for easy cloud access.
  • Arduino-Foosball - Uses a Arduino Uno with Ethernet shield to create a web-enabled Foosball table.

Wireless

Projects that run on, or interact with, wireless sensor nodes.
  • Synapse Power Meter - Uses Synapse Portal + Wireless node to measure AC power and logs to Exosite.
  • Synapse Nodes - Synapse remote and master node python scripts for creating cloud-connected wireless mesh networks.
  • PC Chronos Gateway - Runs on a PC and monitors a Texas Instruments Chronos watch via a CC1111 USB AP. Logs accelerometer and button data to Exosite.
  • CC1111 Chronos Gateway - Firmware for a CC1111 chip that monitors a Chronos watch RF link and sends data out over CC1111 UART.
  • Janus HTTP Gateway - Python code that runs in a Telit cell (GSM) chip on a Janus gateway to send UART data to Exosite.

Chat

Projects that use XMPP to chat with other cloud resources.
  • XMPP Hello - PC app that uses XMPP chat to send "hello world" chat message to Exosite.
  • PC XMPP CPU Monitor - Monitors a PC's CPU activity and logs to Exosite using XMPP chat messages.

PC Apps

Projects that run on a PC.
  • Device Emulator - .NET GUI app that emulates device send/receive functionality.
  • Python Hello World - Super simple python app that sends "hello world" to Exosite using the HTTP Data Interface.
  • PC Serial Gateway - Monitors a PC serial port for data and sends/receives data to/from Exosite based on the UART stream.
  • PC XMPP CPU Monitor - Monitors a PC's CPU activity and logs to Exosite using XMPP chat messages.

Scripts

Device simulation scripts
  • Device Provisioning and Management - Python scripts demonstrating how to provision and manage devices with the Provision API.
  • TCP Read and Write - Python script that uses the simple HTTP data interface to read and write data.
  • UDP Write - Python script that uses the UDP single shot API to write data.
Helper scripts
  • Archive Data - Python app that archives data in date range to a .csv file. Download data from the cloud.
Portal widget scripts
  • Custom Widget Scripts - JavaScript apps that uses user's data from the cloud to represent it in a number of different ways.
